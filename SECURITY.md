# Security Policy

## Supported versions

Only the latest published version of `pi-extension-settings` and its SDK
(`@axnic/pi-extension-settings`) receives security fixes.

| Version | Supported |
| ------- | --------- |
| latest  | yes       |
| older   | no        |

## Reporting a vulnerability

This is a personal, hobby project maintained on a best-effort basis. There are
no guaranteed response timelines or SLAs.

Please **do not** open a public GitHub issue for security vulnerabilities.
Instead, use GitHub's private vulnerability reporting:

1. Go to the [Security tab](https://github.com/axnic/pi-extension-settings/security)
   of this repository.
2. Click **"Report a vulnerability"**.
3. Fill in the details — affected version, reproduction steps, potential impact.

Reports are handled privately. Once a fix is available it will be released and
the advisory will be published.
